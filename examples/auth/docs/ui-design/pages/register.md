## 1. 页面概述

注册页面允许新用户创建账户。与登录页面保持**完全一致的视觉语言**：

- 相同的蓝紫渐变配色方案
- 相同的渐变背景 + 装饰色块
- 相同的卡片圆角、阴影、间距系统
- 居中卡片布局，视觉焦点明确
- **现代感与品质感**：渐变按钮、柔和阴影、精致排版

## 4. 表单验证规则

| 字段 | 规则 | 错误提示 |
|------|------|----------|
| 用户名 | 必填，3-20字符，仅字母数字下划线 | "用户名长度为3-20个字符" |
| 邮箱 | 必填，有效邮箱格式 | "请输入有效的邮箱地址" |
| 密码 | 必填，至少8位，含大小写和数字 | "密码至少8位，需包含大小写字母和数字" |
| 确认密码 | 必填，与密码一致 | "两次输入的密码不一致" |
| 服务条款 | 必须勾选 | "请阅读并同意服务条款" |

**错误状态视觉：**
- TextField `error` 边框色：`#ef4444`
- helperText 颜色：`#ef4444`
- 服务端错误使用 `Alert severity="error"` 显示在表单上方

## 7. 响应式设计

与登录页完全一致。注册表单因字段更多，在小屏幕上需确保：
- 虚拟键盘弹出时页面可滚动
- 4 个输入框 + 复选框 + 按钮全部可见无遮挡
- 卡片内间距在 xs 断点适当缩小 (`py: 4, px: 3`)
