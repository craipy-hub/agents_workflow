## 3. 组件清单

### 3.1 输入框（4个）

| 字段 | label | placeholder | 特殊属性 |
|------|-------|-------------|----------|
| 用户名 | "用户名" | "请输入用户名" | autoFocus |
| 邮箱 | "邮箱" | "请输入邮箱地址" | type="email" |
| 密码 | "密码" | "请输入密码（至少8位）" | 可见性切换 |
| 确认密码 | "确认密码" | "请再次输入密码" | 可见性切换 |

所有输入框：MUI `TextField` variant="outlined", fullWidth

### 3.2 服务条款复选框

- MUI `Checkbox` size="small" + `FormControlLabel`
- 文案："我已阅读并同意 **服务条款** 和 **隐私政策**"
- "服务条款"和"隐私政策"为 `Link` 组件，primary 色

### 3.3 注册按钮

- MUI `Button` variant="contained" color="primary" size="large" fullWidth
- 文案："注  册"

### 3.4 第三方注册按钮

与登录页面一致，文案改为"使用 Google 注册"/"使用 GitHub 注册"

### 3.5 登录链接

- 文案："已有账户? **立即登录**"
- "立即登录" 为 `Link`，primary 色 + Semi Bold

## 4. 表单验证规则

| 字段 | 规则 | 错误提示 |
|------|------|----------|
| 用户名 | 必填，3-20字符，仅字母数字下划线 | "用户名长度为3-20个字符" |
| 邮箱 | 必填，有效邮箱格式 | "请输入有效的邮箱地址" |
| 密码 | 必填，至少8位，含大小写和数字 | "密码至少8位，需包含大小写字母和数字" |
| 确认密码 | 必填，与密码一致 | "两次输入的密码不一致" |
| 服务条款 | 必须勾选 | "请阅读并同意服务条款" |

---

## 5. 与登录页面的一致性

- 相同的灰色背景 + 白色卡片布局
- 相同的 Logo 占位符
- 相同的第三方登录按钮样式
- 相同的 MUI 主题色值
- 底部互相链接（登录 ↔ 注册）
